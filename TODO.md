## 🔧 ТРЕБУЕТСЯ ДЕПЛОЙ НА СЕРВЕР

### 1. Обновить файлы на сервере

```bash
cd /var/www/oncomind

# Обновить из репозитория
git pull origin main

# Проверить файлы
ls -la backend/knowledge_base/

# Перезапустить backend
systemctl restart oncomind

# Проверить статус
systemctl status oncomind
```

### 2. Проверить работу

```bash
# Проверка API рекомендаций
curl http://127.0.0.1:5000/api/guidelines

# Проверка конкретной рекомендации
curl http://127.0.0.1:5000/api/guidelines/breast-cancer-2026

# Проверка поиска
curl "http://127.0.0.1:5000/api/guidelines?q=рак"
```

### 3. Проверить в браузере

1. Откройте https://oncomind.ru/doctor/dashboard.html
2. Войдите как `doctor@oncomind.ai` / `Doctor123!`
3. Проверьте раздел "Клинические рекомендации"
4. Нажмите "Читать" - должна открыться рекомендация в модальном окне
5. Проверьте поиск - введите "рак" или "меланома"
6. Нажмите "Показать все" - должны показать все 12 рекомендаций
